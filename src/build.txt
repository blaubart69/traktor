prerequistites
--------------

    openCV:
        + sudo apt install libopencv-dev

    pistache:
        + git clone https://github.com/pistacheio/pistache.git
        + bauen
        + gebautes lib, include --> ./deps/pistache/
        + src von pistache      --> ./deps/pistache/    (wegen code completion)


cmake, make, run
----------------

    mkdir build
    cd    build
    cmake ..
    make


build release Version
---------------------
    mkdir Release
    cd Release
    cmake -DCMAKE_BUILD_TYPE=Release ..
    make
